环境变量配置

# 环境变量配置

所有的 HOME 类型变量都不要加分号，因为要%HOME%/bin 到 path 中

# 一、jdk 环境变量：

系统变量：

新建系统变量 CLASSPATH 复制： .;%JAVA_HOME%\lib\dt.jar;%JAVA_HOME%\lib\tools.jar;

新建系统变量 JAVA_HOME 复制： D:\Program Files\Java\jdk

编辑 Path(加在最前面) %JAVA_HOME%\bin;%JAVA_HOME%\jre\bin;

测试 Path 环境变量是否配置成功： java -version

测试 CLASSPATH 环境变量是否配置成功 javac
