# 图片

  • 1.命名全部用小写英文字母||数字||_的组合，其中不得包含汉字||空格||特殊字符

  • 2.运用 css 精灵图技术集中小的背景图或图标，减小页面 http 请求，但注意，请务必在对应的精灵图 psd 源图中划参考线，并保存至 img 目录下

  • 3.所有页面元素类图片均放入 img 文件夹，测试用图片放于 demo 文件夹
