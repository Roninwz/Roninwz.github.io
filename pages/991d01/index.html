# Buffer 模块是干什么的

Buffer 类:Buffer 是用与读取或操作二进制数据流的全局变量.

Buffer 大小固定,在被创建的时候确定,无法调整.

Buffer 的使用场景, 非 utf8 字符存储 post 数据 concat.

# Buffer 和字符串区别

  • Buffer 与字符串有一个重要区别。字符串是只读的，并且对字符串的任何修改得到的都是一个新字符串，原字符串保持不变。

至于 Buffer，更像是可以做指针操作的 C 语言数组。例如，可以用[index]方式直接修改某个位置的字节。

  • 字符串转 Buffer

new Buffer(str, [encoding])，默认 UTF-8 buf.write(string, [offset], [length], [encoding])

  • Buffer 转字符串

buf.toString([encoding], [start], [end])

# Stream 是什么，使用的两种模式

流中的数据有两种模式,二进制模式和对象模式. 二进制模式, 每个分块都是 buffer 或者 string 对象. 对象模式, 流内部处理的是一系列普通对象.

# Stream 的几种类型

  • Readable (可读流)
  • Writable (可写流)
  • Duplex (读写流)
  • Transform (转换流)
