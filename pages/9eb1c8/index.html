# jdk 安装配置

  • 下载地址

    https://www.oracle.com/cn/java/technologies/javase/javase-jdk8-downloads.html (opens new window)

  • 上传下载的 jdk-8u261-linux-x64.tar.gz 到临时文件 temp

  • 解压 tar -zxvf jdk-8u261-linux-x64.tar.gz

  • 复制解压文件到 sudo mv jdk1.8.0_261 /usr/local/lib/

  • 设置环境变量 sudo vi /etc/profile

export JAVA_HOME=/usr/local/lib/jdk1.8.0_261
export JRE_HOME=${JAVA_HOME}/jre
export CLASSPATH=.:${JAVA_HOME}/lib:${JRE_HOME}/lib
export PATH=.:${JAVA_HOME}/bin:$PATH
