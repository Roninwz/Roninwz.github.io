# excel 技巧

# 插入下拉菜单

  • 选中单元格
  • 数据-->数据验证-->数据验证
  • 设置-->将允许条件设置为序列
  • 来源中输入 内容如 1,2,3,4（英文逗号）

# 自动求和

  • 选中需要求和的行或者列

  • 开始-->自动求和（右侧编辑栏）

    Alt + =

# 快速复制

  • 选中需要赋值的单元格
  • ctrl+D

# 日期下拉递增

  • 按住 ctrl 键下拉
